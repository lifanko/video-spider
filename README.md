# video-spider
An easy video-spider that Using PHP

### Feature：
 + 无须数据库
 + 任意级目录配置
 + 全自动爬取&内容更新

### Linux Server:
PHP7 + Apache/2.4.6 (CentOS)
##### Setup:
将文件复制进应用目录，默认文档设为‘index.php’即可。
##### Attention:
无法在Linux服务器上实现命名空间类的自动加载，请在文件中引入所需类。

### Windows Server:
PHP5.6.28 + IIS6.2 (Windows Server 2012)
##### Setup:
将文件复制进应用目录，默认文档设为‘index.php’即可。
